# TÀI LIỆU YÊU CẦU PHẦN MỀM (SRS)

## 1. Project Overview
The HelpDesk System is designed to streamline customer support operations by providing a centralized platform for customers to submit support requests and for agents to manage, track, and resolve these requests efficiently. The Minimum Viable Product (MVP) will focus on core functionalities including ticket submission, status tracking, and basic agent management.

## 2. User Stories
### US001 - Submit New Support Ticket
As a customer, I want to submit a new support ticket with a subject, description, and my contact information so that I can report an issue or ask a question.

### US002 - View My Ticket Status
As a customer, I want to view a list of all my submitted tickets and their current status so that I can track their progress without needing to contact support directly.

### US003 - View Open Tickets Dashboard
As a support agent, I want to view a dashboard showing all open and unassigned tickets so that I can quickly identify new issues that need attention.

### US004 - Assign Ticket to Self
As a support agent, I want to be able to assign an unassigned ticket to myself so that I can take ownership and begin working on resolving the issue.

### US005 - Update Ticket Status
As a support agent, I want to update the status of an assigned ticket (e.g., 'Open', 'In Progress', 'Resolved', 'Closed') so that customers and other agents are aware of its current state.

### US006 - Add Internal Ticket Notes
As a support agent, I want to add internal notes to a ticket so that I can record troubleshooting steps, relevant information, or communicate with other agents about the issue without customer visibility.

### US007 - Receive Ticket Submission Confirmation
As a customer, I want to receive an email confirmation when my ticket is submitted so that I know my request has been received.
