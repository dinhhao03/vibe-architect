2. 🏗️ KIẾN TRÚC HỆ THỐNG (TECHNICAL ARCHITECTURE)

2.1 8-Tier Cascading LLM

Cơ chế tự động chuyển đổi thông minh giúp hệ thống vượt qua mọi rào cản về hạn mức API:

Pro (2.5 -> 1.5) -> Flash (2.5 -> 2.0 -> 1.5) -> Lite (2.5 -> 2.0) -> Mock

2.2 Response Caching (MD5)

Mọi kết quả thành công đều được lưu bám (Cache) tại .cache/. Lần sau chạy cùng câu lệnh, kết quả hiện ra ngay lập tức với 0ms đễ và 0 API calls.

2.3 Multi-Key Rotation

Hệ thống tự động luân phiên sử dụng tối đa 10 API keys khác nhau để tối đa hóa hạn mức miễn phí từ Google AI Studio.
